通过 SSH 连接 git 服务器

2023-06-13 · 1 min · 45 words · Ivan

[!Tip] 众所周知，SSH 连接 git 的速度是最快的，所以这里拿 github 为例，记录一下操作过程。

先在本地生成证书：

ssh-keygen -t rsa -b 4096 -C "ivanusong@qq.com"

或者直接使用 ssh-keygen 一路回车不改参数也可。

把生成好的证书的公钥部分比如 id_rsa.pub 文件中的文本内容复制出来，粘贴到 github 的 ssh 设置中。

cat id_rsa.pub # 输出后复制文本

之后设置本地的 ssh 配置文件，将以下内容加入进配置中，其中证书路径要使用你的私钥：

Host github.com
  HostName github.com
  User git
  IdentityFile ~/.ssh/id_rsa

最后使用 ssh -T git@github.com 命令测试一下连接，看到下方提示则配置成功：
